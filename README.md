# Data analysis tutorials

Short, hands-on tutorials on reproducible data analysis.

## Tidy Data Analysis in Excel — `excel_power_query/`

An introductory **Power Query** workshop: turn several messy reviewer
spreadsheets into one tidy, refreshable table.

- `power_query_workshop_slides.pdf` — the workshop slides
- `writeup.pdf` — companion reference handout
- `data/` — the four example spreadsheets

**Get the files:** click the green **Code** button above → **Download ZIP**,
then unzip. No GitHub account needed.

Requires Excel for Windows (2016 or newer); Power Query is built in.
